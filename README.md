# Canteen Face Recognition System

## Project Overview

The Canteen Face Recognition System is a prototype web application developed using Python, Flask, OpenCV, and DeepFace. The system automatically identifies employees through facial recognition and records meal attendance without manual registration.

This project was developed as an internship prototype to demonstrate facial recognition integration, employee management, and meal attendance tracking.

---

## Features

* Role-Based Login System (Admin, HR, Canteen Staff)
* Professional Dashboard UI
* Employee Management
* Employee Search Functionality
* Face Recognition using DeepFace and OpenCV
* Automatic Meal Attendance Recording
* Meal Records Management
* CSV Report Export
* Local JSON Data Storage Prototype
